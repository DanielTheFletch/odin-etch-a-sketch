How to use

Below is a brief guide on how to use the various tools and features of Etched & Sketched.

  • Pen tool  ( )  — When selected, click any square to fill it with the selected color. Click and drag to color multiple squares quickly.
  • Eraser tool  ( )  — When selected, click any square to reset it to its default white color. Click and drag to erase multiple squares quickly.
  • Replace tool  ( )  — When selected, click any square to fill it and all matching squares with the selected color.
  • Change color  ( )  — Change the selected color for use with the Pen and Replace tools. Defaults to black.
  • Change grid size  ( )  — Change the size of the canvas to any square size between 1x1 and 100x100. Defaults to 16x16.
  • Clear canvas  ( )  — Remove all content from current canvas. Cannot be undone.
  • Help  ( )  — Open this help dialog.
  • Undo  ( )  — Undo a previous Pen, Eraser, or Replace action.
